# Arbor Website

Simple static marketing site for Arbor.

## Local Preview

From repo root:

```bash
cd website
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Deploy

You can deploy the `website/` directory directly to:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Any static file host

Current live deployment: [https://penso.github.io/arbor/](https://penso.github.io/arbor/)
